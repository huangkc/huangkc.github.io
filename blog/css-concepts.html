CSS Concepts

Margin, Border, and Padding

9/2/2015

HTML elements can be positioned through the use of margin, border, and padding in CSS. This allows you to control how your web page looks. You can think of HTML elements, whether they are text or pictures, as being contained in boxes. We can to add a border around elements and define the space between elements. The Box Model looks like this:


Image courtesy of W3Schools.com.

In this figure, content is the text or picture we are try to position and it is surrounded by a border. Padding is the space between the content and the border, and margin is the space around the element holding the content. By adjusting the margin, we can move HTML elements relative to other elements and relative to the edge of the screen. Similarly, we can adjust the padding to move the border relative to the content. You can specify the size, style, and color of the border property like this:

{
border: width, style, color;
}

For padding and margin, you can specify values for the top, right, left, and bottom:

{
padding-top: some value;
padding-right: some value;
padding-bottom some value;
padding-left: some value;

margin-top: some value
margin-right: some value
margin-bottom: some value
margin-left: some value
}

Notice the number of variables that can be adjusted individually. This gives you very fine control of of how your page looks. Clearly there is a lot more to margin, border, and padding in CSS. Hopefully this quick look gave you a helpful introduction to these concepts. To learn more about the Box Model and CSS positioning checkout this link.
