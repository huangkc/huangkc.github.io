Bucket List
An application built in Meteor with the React.js library. It allows users to create, organize, and maintain their own bucket list while including a searchable map provided through the Google Maps API should they want to search items related to their bucket list item.
My Auction Site
A web application that simulates an online auction where users can place bids on items and manage their own inventory. Built on Sinatra with Ruby backend and Postgres database.
Stumblrrr
A web application that helps users find interesting things to do in San Francisco when they have free time for a detour. Built with Ruby on Rails backend and Javascript frontend. Submitted to the Dev Bootcamp Civic-Hackathon in January 2016.
